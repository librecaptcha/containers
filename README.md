# SBT on Alpine

This is a base docker image used in the LibreCaptcha project. 
It is based on Alpine and has specific versions of scala, sbt and jre installed to support the project.

## Current Versions

* SBT: `1.3.13`
* Scala: `2.12.10`
* Docker: `19.03.12`
* Java: `OpenJDK11-jre`
* Alpine: `3.12.4 (latest)`
